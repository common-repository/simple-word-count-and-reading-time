== Description ==

Display word count and reading time on your article with a nice and clean like YouTube progress bar. 

== Installation ==
1. Upload `Word-Count-and-Read-Time` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Simple Word Count and Reading time can be accessed under the "Dashboard" Menu of the WordPress Admin.

Or you can search in Dashboard > Plugins > Add New

== Frequently Asked Questions ==

= How it works? =
This plugin works automatically, you don't hae to do anything. Just install it and it will update the reading time on all your current articles.

= Where is the data saved? =
The data: formated reading time, raw reading time and raw word count are saved on post meta.
